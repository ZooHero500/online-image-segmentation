 在线图片分割网站

 1. 上传图片
 2. 用户拖动分割线（最好有磁吸效果）
 3. 有横向和竖向的分割线
 4. 点击生成，分割图片
 5. 下载分割后的图片


技术架构：
1. 前端随便，可以使用NextJS，如果有对应开源的库支撑分割等功能就最好，不用重复造轮子。
2. css 使用 shadcn ui + tailwindcss + Heroui
3. 使用 bun


整体界面：（有点类似 Figma 的操作界面）
侧边，操作的历史记录，展示上传过的图片，点击有历史记录
主界面，上传图片，拖动分割线，点击生成，下载分割后的图片


所有数据存储到浏览器本地，可以考虑存储到 local storage 或者 indexdb，至于选择哪一个，看数据量大小。
